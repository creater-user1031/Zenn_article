# 入力した願い事を取得して、結果に表示させる。

以前の回で、**textarea**タグで入力する箇所を作りました。
また、**fortune**という空のdivタグを用意していましたね。

今回は、これら2点の個所を利用して
**textarea**に入力した内容を、**fortune**内に再度はめ込む動きを実装しました。

## HTMLファイルがJSファイルを読み込めるようにする。

まず、構成のひな型であるHTMLとは別に、
動きをつけるJSファイルを用意します。

その後、HTMLファイルでbodyの締めタグ直前に

``` html
<script src="script.js"></script>
```

と記載し、jsファイルに書いたコードを読み込めるようにし、準備が整いました。

## 本題の動きをするためのJSコード

以下が実際に実装したJavaScriptコードです。

```js
const button = document.getElementById("drawButton");
const fortune = document.getElementById("fortune");
const wishInput = document.getElementById("wishInput");

button.addEventListener("click", function () {
    const wish = wishInput.value;

    fortune.textContent = "あなたの願い：「" + wish + "」";

    if (wish === "") {
        fortune.textContent = "願い事が記載されていないため、おみくじを引くことができませんでした。";
        return;
    }
})
```

- 初めに、constでhtml上の各種IDタグと、jsファイル内での単語の紐づけをします。

constで定義した変数は再代入できないため、
意図しない変更を防ぐことができます。

ちなみにconstとは異なり、柔軟に定義を後から変更できるものが「let」です。

- **button**を押したときに起こる動きの構築

「おみくじを引く」というボタンをクリックしたときに、もし値が入力されていたら（何らかの願い事が入力されていたならば）、入力されているお願い事を結果として再度表示し、値が何も入っていない場合は「願い事が記載されていないため、おみくじを引くことができませんでした。」と返ってくる仕組みです。
